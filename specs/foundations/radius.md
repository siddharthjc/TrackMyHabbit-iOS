# Border radius

| Token | Value | Swift |
|-------|------:|--------|
| `--radius-sm` | 8px | `AppTheme.Radius.sm` |
| `--radius-md` | 12px | `AppTheme.Radius.md` |
| `--radius-lg` | 20px | `AppTheme.Radius.lg` |
| `--radius-xl` | 24px | `AppTheme.Radius.xl` |
| `--radius-2xl` | 30px | `AppTheme.Radius.xl2` |
| `--radius-pill` | 56px | `AppTheme.Radius.pill` |
| `--radius-glass` | 28px | `AppTheme.Radius.glass` |
| `--radius-full` | 999px | `AppTheme.Radius.full` |

## Usage

- **Cards and sheets:** `--radius-xl` / `Radius.xl`
- **Pill CTA / large chrome:** `--radius-pill` / `Radius.pill`
- **Glass bar:** `--radius-glass` / `Radius.glass`
